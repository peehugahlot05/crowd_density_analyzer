About This Project

The Crowd Density Analyzer is a smart web application built using computer vision and deep learning techniques to help monitor and manage crowd density in public spaces. It is ideal for use in:

  • Events & Gatherings
  • Public Transport Stations
  • Shopping Malls
  • Disaster Management Zones

Our app uses YOLOv8 for real-time person detection and CSRNet for accurate crowd density estimation. These models complement each other â€” YOLO detects and counts visible people, while CSRNet estimates the total density, including occluded or partially visible individuals.
