How It Works

This app uses two deep learning models to detect and estimate crowd levels from images:

YOLOv8

Detects people in images using bounding boxes. Fast and real-time object detection.

CSRNet

Generates density maps to estimate crowd count, even when individuals are not clearly visible.

Steps to Use

  1. Go to the Home page.
  2. Upload an image of a crowd scene.
  3. Click Analyze to run YOLO and CSRNet.
  4. View bounding boxes, heatmaps, and alert if high density is detected.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Models: YOLOv8 (Ultralytics), CSRNet (PyTorch)
  • Deployment: Render
